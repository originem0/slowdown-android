## 这是什么？

SlowDown 是一款帮助你觉察手机使用的 Android 应用。

**它不会在你打开应用时拦截你**。它等待着，在你刷了 10 分钟、20 分钟、30 分钟后，在你已经陷入无意识滑动的时候，温柔地弹出一个深呼吸界面，让你暂停几秒。这短短的几秒，足以让你意识到时间在流逝，然后做出更好的选择。

**不是阻止你开始，而是帮你在沉浸中清醒。**

---

## 为什么选择 SlowDown？

### 与其他限时应用的区别

| 特点 | SlowDown | 其他应用 |
|------|----------|----------|
| **干预时机** | 使用过程中定期提醒 | 打开时就拦截 |
| **干预方式** | 深呼吸等待，温柔提醒 | 直接锁死，强制禁用 |
| **隐私保护** | 数据全部存在本地，不联网 | 通常需要注册账号、云同步 |
| **使用体验** | 尊重用户选择，可随时继续 | 常有"绕过限制"的挫败感 |
| **设计理念** | 在沉浸中唤醒觉察 | 把用户当敌人 |

### 核心理念

- **过程中觉察**：问题不是"你该不该打开"，而是"你刷了多久还没意识到"
- **不制造焦虑**：没有恐吓式的"你已浪费 X 小时"，只有平静的深呼吸
- **尊重选择**：深呼吸后可以继续使用，决定权始终在你手中

## 功能介绍

### 使用过程中的深呼吸提醒

SlowDown 会在后台追踪你的使用时间，当达到设定的阈值时弹出深呼吸界面：

- 使用达到每日限额的 **80%** 时：温柔提醒，让你意识到时间
- 使用达到 **100%** 时：根据模式决定是提醒还是强制暂停
- 每次提醒后有 **冷却时间**，不会频繁打扰你

深呼吸界面：
- 跟随圆圈深呼吸
- 倒计时结束后可选择继续或离开
- 可设置跳转到替代应用（比如：刷够了抖音 → 去看书）

### 使用时间限制

为每个应用设置每日使用上限：

| 模式 | 说明 |
|------|------|
| **仅统计** | 只统计时间，不做干预 |
| **温和提醒** | 达到限额后提醒，但可继续使用 |
| **严格限制** | 达到限额后强制停止，今日无法继续 |
| **完全禁止** | 打开即阻止，无法使用 |

### 短视频模式

针对抖音、B站等刷视频应用，开启后每 30 秒主动检查使用状态，确保深呼吸提醒能正常触发。

### 使用统计

- 今日拦截次数
- 节省时间
- 觉察率（选择放弃使用的比例）
- 本周使用趋势
- 最常拦截的应用

### 其他特性

- 支持中文/英文切换
- 所有数据本地存储，保护隐私
- 支持 Android 系统备份
- 简洁美观的界面设计

## 如何使用

### 1. 安装并授权

安装后需要开启以下权限：

**必要权限**（核心功能）

| 权限 | 作用 |
|------|------|
| 无障碍服务 | 检测应用启动 |
| 悬浮窗权限 | 显示深呼吸界面 |
| 使用情况访问 | 读取应用使用时间 |

**建议权限**（提升稳定性）

| 权限 | 作用 |
|------|------|
| 忽略电池优化 | 防止服务被系统杀死 |

**MIUI 用户额外设置**

小米手机需要额外开启「后台弹出界面」权限，否则深呼吸界面无法正常显示。

### 2. 添加监控应用

1. 打开 SlowDown，进入「应用」页面
2. 点击想要监控的应用
3. 设置限制模式和每日时间限额

### 3. 开始使用

设置完成后，正常使用手机即可。SlowDown 会在后台默默追踪使用时间，**只有当你使用到一定时长时才会提醒你**。不是在你打开应用的瞬间拦截，而是在你"刷着刷着忘了时间"的时候轻轻提醒。

## 常见问题

**Q: 为什么不直接锁死应用？**

A: 强制戒断容易引发逆反心理。SlowDown 的目标是帮你建立觉察，而不是和你作对。当你能意识到自己在做什么，自然会做出更好的选择。

**Q: 等待时间可以调整吗？**

A: 可以，在设置中调整默认倒计时秒数（3-30秒）。

**Q: 会影响手机性能吗？**

A: 不会。SlowDown 采用被动监听方式，只在你打开应用时才工作，不会持续占用资源。

**Q: 我的数据安全吗？**

A: 所有数据都存储在你的手机本地，不会上传到任何服务器。支持 Android 系统备份功能。

**Q: 刷短视频时为什么没有提醒？**

A: 短视频应用滑动切换视频时不会触发常规检测。请在应用设置中开启「短视频模式」，启用主动检查。
